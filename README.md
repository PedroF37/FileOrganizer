# FileOrganizer
## Projeto de organizador de arquivos em C

* Dada uma pasta com arquivos de várias extensões, programa cria pastas com o nome da extensão e move arquivos para a pasta. Arquivos .txt para pasta Txt arquivos .pdf para pasta Pdf etc.
* Programa faz a recursão nas pastas
*  Usa a biblioteca utils que está em: `https://github.com/PedroF37/Utils`
